## 🎯 แพ็กที่แนะนำ: Custom — เว็บการตลาด/เซลเพจ SEO เต็มระบบ
### ราคา: **15,000 บาท** (เริ่มต้น · ปรับตามขอบเขตจริง)

**ทำไมแพ็กนี้:** ตอบโจทย์ "ขายจบในเว็บ ครบทุก SKU แต่ไม่จำเจ" + "เน้น SEO" ได้ในหน้าเดียว และเป็นฐานที่ต่อยอดเป็นระบบขายเต็มได้ภายหลัง

> **demo นี้ = template "Clinical Dark" (flagship แพงสุด)** — ดีไซน์ใหม่ทั้งหมด ไม่ซ้ำงานเดิม · ใช้ **สีจากโลโก้แบรนด์จริง** (cyan #17D9FC + navy) · **ภาพจริงทั้งหมดจากเว็บลูกค้า** (สินค้า/รางวัล THBA 2023/งานวิจัยญี่ปุ่น-ไทย/before-after) → จุดขายคือ "เห็นภาพจริงว่าแบรนด์คุณบนเว็บใหม่เป็นแบบนี้"

**ได้อะไร:**
- เซลเพจเชิงเล่าเรื่อง (editorial sale page) — flow: ปัญหา → วิธีแก้ → ส่วนผสม → รีวิว → เลือกสินค้า → ปิดการขาย
- โชว์ครบ **ทุก SKU (10 รายการ)** จัดกลุ่ม เซรั่ม / แชมพู-มาส์ก / เซ็ต · ปิดการขายผ่าน LINE + โทร
- **SEO เต็มระบบ:** title/meta/description, Open Graph + Twitter (การ์ดแชร์ FB/LINE), JSON-LD (Organization + Product + FAQPage), heading hierarchy, alt, sitemap.xml, robots.txt, ปรับ performance
- responsive ครบทุกอุปกรณ์ (มือถือ/แท็บเล็ต/เดสก์ท็อป) + QA วัด DOM จริง
- โครงรองรับ **รีแบรนด์** — เปลี่ยนสี/ฟอนต์/โลโก้/ภาพได้โดยไม่ต้องรื้อ
- โทนดีไซน์พรีเมียมสำหรับผู้ชาย 40–60 (ดำถ่าน–ทองบรอนซ์–เขียวสมุนไพร)

## 🔧 ดูแลรายเดือน (คิดตามต้นทุนจริง — ไม่ตั้งขาดทุน)
- **Host:** static host ฟรีได้บน Cloudflare Pages → ลูกค้าจ่ายแค่ค่าโดเมน
- **Care + SEO รายเดือน:** อัปเดตเนื้อหา/ราคา/สินค้าใหม่, ติดตามอันดับ, ปรับจูน on-page

# ราคา + แผนเสนอขาย — บริษัท บุษราคัม กรุ๊ป จำกัด

> ประเภท: ผู้พัฒนาอสังหาริมทรัพย์ (อาคารพาณิชย์ / โฮมออฟฟิศ) · 2 โครงการ
> บริบท: **ลูกค้าเก่า** (เคยจ่าย 9,000) · เคยเสนอ 35,000 → ไม่ไหว · ตอนนี้โดเมนล่ม ไม่มีคนดูแล
> โจทย์: ฟื้นเว็บลุคใหม่ใน **งบที่จับต้องได้** + แก้ปมจริง = "ไม่มีคนดูแลจนเว็บหาย" → ขายดูแลรายเดือน

## Tier ที่แนะนำ: **เว็บธุรกิจ — 6,900 บาท** (ฟื้นเว็บ ลุคใหม่)

**เหตุผล:**
- เป็นผู้พัฒนาอสังหาฯ ที่มี **หลายโครงการ + จุดขายเรื่องทำเล** → ต้องการเว็บหลายส่วน (hero, โครงการ, ทำเล, ติดต่อ) ไม่ใช่หน้าโปรไฟล์เดียว → เกินระดับ 3,900
- Custom 15,000 ยังไม่จำเป็นในเฟสแรก (ยังไม่ต้องมีระบบจองยูนิต/ฐานข้อมูลยูนิตเรียลไทม์)
- **6,900** ให้โครงครบตามเดโม (โชว์ 2 โครงการ, จุดขายทำเล, use-case, ขั้นตอนซื้อ, ติดต่อ) — คุ้มและพร้อมใช้จริง
- จุดสำคัญทางจิตวิทยา: ลูกค้าเคยช็อกกับ 35,000 → ตัวเลข **6,900 ฟื้นได้เลย** ทำให้ตัดสินใจง่ายขึ้นมาก

### สิ่งที่ได้ในแพ็กเกจ 6,900
- เว็บหน้าเดียวแบบในเดโม (one-page) ปรับเนื้อหา/สี/รูปจริงของทั้ง 2 โครงการ
- mobile-first โหลดไว + SEO meta + OG (แชร์ใน FB/LINE แล้วการ์ดขึ้นสวย)
- ปุ่มโทร / แผนที่ Google / นัดชมโครงการ ครบ
- โลโก้แบรนด์ + จุดขายทำเล (ม.มหิดล / เซ็นทรัล / แม็คโคร / โลตัส)
- ขึ้นโฮสต์ให้ (Cloudflare Pages — ฟรี) + **ช่วยกู้/ผูกโดเมน busarakhumgroup.com กลับมา**

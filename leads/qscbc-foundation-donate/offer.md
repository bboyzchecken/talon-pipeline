# Offer — เว็บไซต์รับบริจาค QSCBC Foundation (donate-only)

> ข้อเสนอ **ปรับขอบเขต** จากสัญญาเดิม (donate + ecommerce) → **เน้นระบบรับบริจาคก่อน**
> ลูกค้าเป็นมูลนิธิเฉลิมพระเกียรติ — งานต้องสง่างาม น่าเชื่อถือ โปร่งใส

## แพ็กเกจที่แนะนำ

### ⭐ เว็บไซต์รับบริจาค (Custom Donation Platform) — เริ่มต้น 15,000 บาท
เว็บโดเนทโฉมใหม่ ดีไซน์เฉพาะมูลนิธิฯ คงตราสัญลักษณ์/สี/ฟอนต์เดิม 100%
- หน้าเดียวจบ ลื่นไหล (hero · พันธกิจ · เลือกบริจาค · ช่องทางธนาคาร · โครงการ · ติดต่อ)
- โมดูลบริจาค: เลือกจำนวน + *ครั้งเดียว/รายเดือน* + กรอกข้อมูลผู้บริจาค
- Responsive เต็มรูปแบบ + SEO + OG (การ์ดแชร์ FB/LINE ขึ้นถูกต้อง)

**เหตุผล:** เป็นงานออกแบบเฉพาะ + ต้องเชื่อมระบบชำระเงิน จึงเป็น tier Custom ไม่ใช่เว็บโปรไฟล์สำเร็จรูป

## Add-on ที่เกี่ยวข้อง (เลือกตามต้องการ)

| Add-on | รายละเอียด | ราคา (ประเมิน) |
|---|---|---|
| **เชื่อมระบบชำระเงิน** | บัตรเครดิต (KTC paygate เดิม / หรือ gateway ใหม่ เช่น Omise/2C2P) + ยืนยันผล | 8,000–15,000 * |
| **บริจาครายเดือน (recurring)** | ตัดบัตรอัตโนมัติทุกเดือน + จัดการสมาชิกผู้บริจาค | 6,000–10,000 * |
| **ใบเสร็จ/ใบอนุโมทนาอัตโนมัติ** | ออก PDF + ส่งอีเมลให้ผู้บริจาคเพื่อใช้ลดหย่อนภาษี | 5,000–8,000 |
| **หลังบ้านจัดการบริจาค (Dashboard)** | ดูยอด/รายชื่อ/สรุปรายเดือน · export Excel | 8,000–12,000 |
| **2 ภาษา (ไทย/อังกฤษ)** | สลับ TH/EN ทั้งเว็บ | 4,000 |

> \* ขึ้นกับ payment gateway ที่มูลนิธิฯ เลือกและค่าธรรมเนียมของผู้ให้บริการ — สรุปแน่นอนหลังเลือก gateway

## ดูแลรายเดือน (เลือกได้)
- **Basic — 800 บาท/เดือน:** โฮสต์ + โดเมน + สำรองข้อมูล + แก้ไขเล็กน้อย (ต้นทุนจริงต่ำ เพราะ host หน้าเว็บแบบ static ได้ฟรีบน Cloudflare จ่ายแค่โดเมน + ดูแลระบบชำระเงิน)
- **Care — 1,500 บาท/เดือน:** เพิ่มอัปเดตเนื้อหา/โครงการ + ดูแล payment + รายงานยอดบริจาครายเดือน

> โฮสต์หน้าเว็บฟรีได้ (Cloudflare Pages) — ค่าใช้จ่ายจริงอยู่ที่โดเมน + ฝั่งระบบชำระเงิน อย่าตั้งราคาขาดทุน

# Offer — เว็บไซต์ศูนย์ฯ ครบทุกเมนู (retention proposal)

> ข้อเสนอให้ศูนย์ฯ ใช้แพลตฟอร์มของเราต่อ แทนการย้ายไป Wix — เว็บครบทุกเมนู สองภาษา ธีมฟ้า-ทอง
> ลูกค้าเป็นเชื้อพระวงศ์ / ศูนย์การแพทย์ในสังกัด รพ.จุฬาฯ สภากาชาดไทย — งานต้องสง่างาม น่าเชื่อถือ

## แพ็กเกจที่แนะนำ

### ⭐ เว็บไซต์องค์กรหลายหน้า + ระบบบริจาค (Custom) — เริ่มต้น 15,000 บาท
เว็บครบทุกเมนูเทียบเท่าเว็บใหม่ ในธีมและสีของศูนย์ฯ
- โครงสร้างเมนูครบ 4 กลุ่ม (เกี่ยวกับศูนย์ฯ · การบริการ/หน่วยบริการ · โครงการ · บริจาค/ติดต่อ)
- ดีไซน์ฟ้า-ทอง + ตราสัญลักษณ์ + SukhumvitSet
- โมดูลบริจาค (เลือกจำนวน · ครั้งเดียว/รายเดือน) — จุดแข็งที่เว็บ Wix ทำไม่ได้ดีเท่า
- Responsive + SEO + OG + โหลดไว

**เหตุผล:** เป็นเว็บหลายหน้า + ดีไซน์เฉพาะ + ระบบบริจาค = tier Custom

## ดูแลรายเดือน
- **Basic — 800 บาท/เดือน:** โฮสต์ + โดเมน + สำรองข้อมูล + แก้ไขเล็กน้อย *(หน้าเว็บ static โฮสต์ฟรีได้บน Cloudflare จ่ายแค่โดเมน)*
- **Care — 1,500 บาท/เดือน:** เพิ่มอัปเดตข่าว/โครงการ + ดูแลระบบบริจาค + รายงานยอดรายเดือน

> โฮสต์หน้าเว็บฟรีได้ — ต้นทุนจริงคือโดเมน + ฝั่งระบบชำระเงิน อย่าตั้งราคาขาดทุน

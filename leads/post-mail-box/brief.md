# Brief: Post Mail Box (โพส เมล บล็อก)

> เว็บ **บริษัท** (หน้าแรก/company front page) สำหรับ Post Mail Box — ลูกค้าที่จ้างเราทำ **ระบบบริหารจัดการขนส่ง** (deploy อยู่ที่ postmailbox.co.th)
> โจทย์: ทำหน้าเว็บบริษัทไปวางเป็น **หน้าแรกแทนหน้า login เดิม** โดย **คงโลโก้ / สี / สไตล์จากระบบจริง** (ลูกค้าเป็นนักออกแบบ เคร่งเรื่องสี+ธีมมาก)

## ข้อมูลหลัก
- ชื่อแบรนด์: **POST MAIL BOX** (โพส เมล บล็อก) — โลโก้เต็มเขียน "POST·MAIL·BOX APPLICATION"
- ประเภทธุรกิจ: **ตัวแทน/นายหน้ารับส่งพัสดุ-ไปรษณีย์ ในประเทศและระหว่างประเทศ** (ศูนย์รับฝากส่งหน้าร้าน)

## 🧭 โมเดลธุรกิจ (สำคัญ — กำหนดทิศคอนเทนต์ทั้งหน้า) — ยืนยันจากลูกค้า 2026-06-29
- Post Mail Box = **นายหน้า/ตัวแทน** รับฝากพัสดุหน้าร้าน → ชั่ง+คิดค่าส่ง → **ออกใบเสร็จพร้อมเลขพัสดุ** → **จัดส่งจริงผ่านไปรษณีย์ไทย**
- **ลูกค้าไม่ได้ login** — ลูกค้าได้แค่ใบเสร็จ แล้วเอาเลขพัสดุไป **ติดตามที่ไปรษณีย์ไทยเอง**
- **"เข้าสู่ระบบ" = พนักงานหน้าร้าน / แอดมิน เท่านั้น** → ในเว็บลดเป็นลิงก์เล็ก "เข้าระบบพนักงาน" (header/footer/ส่วนระบบ) ไม่ใช่ CTA ลูกค้า
- **เป้าหมายลูกค้า (เจ้าของ): ขยายแฟรนไชส์/สาขาของตัวเอง** → เว็บทำหน้าที่ **โปรไฟล์บริษัท** ที่ดูน่าเชื่อถือ/ขยายได้ · ส่วน "ระบบจัดการ" เฟรมเป็น **"มาตรฐานเดียวทุกสาขา · พร้อมขยายสาขา"** + ปุ่มเบา ๆ "สนใจเปิดสาขา → ติดต่อเรา" (ยังไม่ทำ funnel แฟรนไชส์เต็ม เว้นแต่ลูกค้าขอ)
- เว็บระบบ (live): https://postmailbox.co.th/ (Nuxt SPA · ปัจจุบัน root = หน้า login ระบบ)
- ที่ตั้ง (จาก Google Maps ref): ย่าน **ยานนาวา / พระราม 3 กรุงเทพฯ** — พิกัด `13.6983059, 100.5377601`
  - ลิงก์: https://maps.app.goo.gl/vc3dNZHeXuX33JB38 (place: "โพส เมล บล็อก")

## ⚙️ ฟีเจอร์ระบบจริง (ถอดจากโค้ด pmb-website → ใช้เขียนคอนเทนต์ส่วน "ระบบจัดการ" แบบไม่มั่ว)
โมดูลใน `components/Admin/` + `service/` + `store/` + ไอคอน sidebar:
- จัดการพัสดุ/กล่อง + **ชั่งน้ำหนัก/ขนาด** (box, weighing) → คิดค่าส่งตามจริง
- **สาขา** (Branch) · **สมาชิก/ลูกค้า** (Member) · **สต๊อก/คลัง** (Stock)
- **อัตราแลกเปลี่ยน** (Exchange) → รองรับส่ง **ระหว่างประเทศ**
- **บิล / สลิป / ยกเลิกบิล / รับชำระ** (bill, slip, cancel_bill, cash-register, withdrawal)
- **แสตมป์/ศุลกากร** (stamp, stamp TH) → งานไปรษณีย์/ข้ามแดน
- **แจ้งเตือน** (notifications: mail/stamp/box) · flow แบบ **steps** (store/steps.js)
- ขอบเขต **ในประเทศ + ทั่วโลก** (thailand / worldwide / earth-globe)

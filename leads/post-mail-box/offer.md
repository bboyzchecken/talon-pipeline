# Offer — เว็บบริษัท Post Mail Box (หน้าแรก)

> ลูกค้าปัจจุบัน (จ้างทำระบบขนส่งอยู่แล้ว) · งานนี้ = หน้าเว็บบริษัทไปวางเป็นหน้าแรกของ postmailbox.co.th
> ลูกค้าเป็นนักออกแบบ เคร่งแบรนด์ → จุดขายคือ "ยึดสี/โลโก้/ไอคอนจากระบบจริง"

## แพ็กเกจที่แนะนำ: **เว็บธุรกิจ — 6,900 บาท**
หน้าเดียวจบ (single-page) แต่ระดับ custom: ดึง design system จากระบบจริง + โชว์ตัวระบบ + แผนที่ + responsive ครบทุกจอ

**ทำไม tier นี้ (ไม่ใช่ 3,900):**
- ไม่ใช่เว็บโปรไฟล์สำเร็จรูป — สกัดสี/โลโก้/ไอคอนจากโค้ดระบบจริงมาเป็น design token
- มีส่วน “ระบบมาตรฐานทุกสาขา” (dashboard motif) ที่ต่อยอดจากงานระบบ → **หนุนเป้าหมายขยายแฟรนไชส์** ของลูกค้า (โปรไฟล์ที่ดูน่าเชื่อถือ/ขยายได้)
- เป็น **หน้าแรกของโดเมน production จริง** ไม่ใช่หน้า demo ลอย ๆ
- สื่อโมเดลถูกต้อง: ตัวแทนรับฝากส่ง · ลูกค้าติดตามที่ไปรษณีย์ไทย · login = พนักงานเท่านั้น
- SEO meta + OG + JSON-LD + แผนที่ + ติดตั้งแทนหน้า login เดิม

> เป็นลูกค้าเก่า → เสนอ **bundle กับงานระบบ** หรือคิดแยกก็ได้ แล้วแต่พี่สะดวก

## ดูแลรายเดือน (คิดตามต้นทุนจริง)
- โฮสต์ static **ฟรี** ได้ (Cloudflare Pages) — โดเมน postmailbox.co.th มีอยู่แล้ว
- **Care 400/เดือน** — แก้คอนเทนต์/รูป/เบอร์/โปรโมชันให้ตามรอบ + ดูแลให้ออนไลน์
- หรือพ่วงเข้าสัญญาดูแลระบบเดิม ไม่คิดเพิ่มก็ได้

# ราคา + แผนเสนอขาย — สินพัฒนา ธุรกิจ

> ประเภท: โรงพิมพ์ / โรงงานบรรจุภัณฑ์ B2B (ลูกค้าองค์กร + แบรนด์ใหญ่)
> โจทย์: มรดก 30 ปี → ทำตลาดออนไลน์ให้เข้าถึงง่าย "คุ้มค่า" คือคีย์เวิร์ดที่ลูกค้าต้องการ

## Tier ที่แนะนำ: **เว็บธุรกิจ — 6,900 บาท** (เริ่มต้นคุ้มค่า)

**เหตุผล:** เป็นธุรกิจ B2B ที่ต้องการความน่าเชื่อถือ + หลายหมวดสินค้า + ช่องทางขอใบเสนอราคา
เว็บโปรไฟล์ 3,900 เล็กเกินไป (โชว์ได้แค่หน้าเดียว) ส่วน Custom 15,000 ยังไม่จำเป็นในเฟสแรก
**6,900** ให้โครงครบตามเดโม (hero, สถิติ, เรื่องราวรุ่นสู่รุ่น, สินค้า, สเปก/ราคา, ขั้นตอน, ติดต่อ) — คุ้มและพร้อมใช้จริง

### สิ่งที่ได้ในแพ็กเกจ 6,900
- เว็บหน้าเดียวแบบในเดโม (one-page) ปรับแต่งเนื้อหา/สี/รูปจริง
- mobile-first โหลดไว + SEO meta + OG (แชร์แล้วขึ้นพรีวิวสวย)
- ปุ่มขอใบเสนอราคา / LINE / โทร ครบ
- โลโก้แบรนด์ลูกค้า + ตัวเลขความน่าเชื่อถือ
- ขึ้นโฮสต์ให้ (Cloudflare Pages — ฟรี) + ผูกโดเมนของลูกค้า

## Add-on ที่เกี่ยวข้อง (เสนอเพิ่มได้ตามงบ)
| Add-on | ราคาโดยประมาณ | เหมาะเพราะ |
|---|---|---|
| **แคตตาล็อกสินค้าแยกหมวด** (กล่อง/ถุง/สติ๊กเกอร์ ฯลฯ มีหน้า/รายละเอียดของตัวเอง) | +3,000–5,000 | สินค้าหลายประเภท ลูกค้าอยากดูสเปกก่อนถาม |
| **ฟอร์มขอใบเสนอราคา** (ส่งเข้าอีเมล/LINE — เก็บแบบงาน/จำนวน) | +2,500 | ลด friction รุ่นใหม่อยากสั่งง่าย |
| **หน้าผลงาน/แกลเลอรีงานจริง** | +2,000 | โชว์งานที่เคยทำให้แบรนด์ดัง = social proof |
| **ตั้งค่า Google Business + SEO ท้องถิ่น** | +1,500 | ให้เจอใน Google Maps/ค้นหาง่ายขึ้น |

> เส้นทางอัปเกรด: เริ่ม 6,900 → เพิ่มแคตตาล็อก + ฟอร์มขอราคา = ขยับเข้าใกล้ระดับ **Custom 15,000** แบบจ่ายเท่าที่ใช้

## ออปชันดูแลรายเดือน
- **Care พื้นฐาน — 500 บาท/เดือน:** โฮสต์ (ฟรีบน Cloudflare) + โดเมน + แก้เนื้อหาเล็กน้อย 1 ครั้ง/เดือน + สำรองข้อมูล
- **Care+ — 1,200 บาท/เดือน:** เพิ่มอัปเดตสินค้า/โปรโมชัน + รายงานทราฟฟิกพื้นฐาน
> ต้นทุนจริงต่ำ (static host ฟรี จ่ายแค่โดเมน ~500–800/ปี) — ตั้งราคานี้ไม่ขาดทุน ลูกค้ารู้สึกคุ้ม

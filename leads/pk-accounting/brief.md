## ⚖️ กฎหมายวิชาชีพบัญชี — ต้องระวัง (สำคัญ)
เว็บสำนักงานบัญชีพูดผิดได้เรื่องคุณสมบัติ/ใบอนุญาต → ทำให้เข้าใจผิด/ผิดกฎหมายได้ กฎที่ยึดในเว็บนี้:
1. **ผู้ทำบัญชี** ต้องขึ้นทะเบียนกับ **กรมพัฒนาธุรกิจการค้า (DBD)** ตาม พ.ร.บ.การบัญชี 2543 — เว็บใส่ช่อง "เลขผู้ทำบัญชี/เลขทะเบียน" เป็น **⬜ ให้ลูกค้ากรอกจริง** ห้ามแต่งเลข
2. **การตรวจสอบบัญชี (audit) / เซ็นรับรองงบ** ทำได้เฉพาะ **ผู้สอบบัญชีรับอนุญาต (CPA)** ที่ขึ้นทะเบียนกับ **สภาวิชาชีพบัญชี (TFAC)** เท่านั้น → เว็บ **ไม่อ้างว่าเป็น CPA เอง** เว้นแต่ลูกค้ายืนยัน; ถ้าให้บริการ audit จริงให้เฟรมว่า *"บริการตรวจสอบบัญชีโดยผู้สอบบัญชีรับอนุญาต"* (ประสาน CPA) — ⬜ ลูกค้ายืนยันสถานะ
3. **ห้ามการันตีผล** — ห้ามเขียน "ไม่โดนปรับแน่นอน / คืนภาษีได้ 100% / ผ่านสรรพากรชัวร์" → ใช้คำ "ถูกต้องตามมาตรฐาน · ตรงเวลา · โปร่งใส" แทน
4. **ห้ามแต่งตัวเลข** — ปีประสบการณ์ / จำนวนลูกค้า / % ตรงเวลา = ⬜ ให้ลูกค้ายืนยันก่อนใช้จริง (ในเดโมใส่เป็น placeholder ที่แก้ง่าย)
